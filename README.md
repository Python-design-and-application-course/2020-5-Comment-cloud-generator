# 评论词云生成器项目说明
本项目使用了Python的爬虫技术，从豆瓣爬取指定作品的评论，并绘制成词云，能快捷展示作品评价。

## 文件说明
上传文件包括：源代码，系统开发说明文件、小组展示PPT、以及程序演示视频，其中源代码中包含图形页面、爬取短评以及制作词云图代码。（程序压缩文件已通过qq另外提交）
